Amazing Features

Here are some amazing features of the Omics Playground. More amazingly, you will be able to use them yourself because our platform makes it easy for anyone to use a wide range of the most advanced machine learning algorithms. In the Playground you can freely visualize, analyze and explore your omics data to discover hidden patterns and trends in your data effectively. A platform that is enjoyable and ‘Makes Data Analytics Great Again’.


Expression and enrichment analysis

For differential expression, Omics Playground calculates up to eight different statistical testing methods. For gene set enrichment analysis, we use more than 50’000 gene sets with up to seven different statistical methods. We combine multiple statistical methods using meta-analysis so you can really trust your numbers. We also perform drug enrichment analysis of more than 5’000 drugs, and find drugs (mono and combination) with similar or opposing signatures compared to yours. You can enter custom gene sets to perform enrichment analysis of your own signature.

  • DE methods: DESeq2, edgeR, LIMMA/Voom, LIMMA-trend, t-test.
  • Enrichment methods: GSEA, GSVA, ssGSEA, Fisher exact, rank correlation, camera, fry.
  • Drug enrichment analysis using drug activity profiles from the L1000 database.

Variable importance and biomarker selection

To understand which genes, mutations or gene sets are more influential on the outcome of your interest, the Playground calculates variable importance using state-of-the-art machine learning algorithms. The outcome can be multiple categories (classes) or patient survival. Using multiple algorithm we select the best biomarkers. We then train classifiers to build a prediction model for your data and phenotype of interest.

  • Feature selection algorithms: LASSO, elastic nets, random forests, and extreme gradient boosting.
  • Classifier algorithms: random forest, deep nets.

Special immuno-oncology analytics

We have implemented a special analysis modules for immuno-oncology. You can infer copynumber variation from single-cell RNA-seq expression, analyse differential immuno-genes usage, perform computational immune celltype profiling using state-of-the-art deconvolution algorithms.

  • Deconvolution methods: contrained-NNLS, NNLM, EPIC, rank correlation and DCQ.
  • Immune cell reference databases: LM22, DICE, ImmProt, ImmunoStates.

Advanced batch effects analysis

Omics data are notoriously sensitive to so-called “batch effects” that can make or break your analysis. Within the Playground you can easily identify batch effects with our unique PCA cluster analysis. Quantify batch effects and visually evaluate a variety of batch correction techniques in real time.

  • Normalization methods: quantile normalization, median-centering, TMM, RLE.
  • Batch methods: quantile normalization, NMM, MNN, ComBat, LIMMA.

Systems biology and multi-omics integration

Biology cannot be explained using a single omics type only. To truly understand biology, we need to measure multiple omics types and integrate heterogeneous information at gene, functional and phenotype level. Our ‘multi-omics’ module analyzes gene expression, methylation, copy number and mutation data, and integrates these data using our state-of-the-art multilayer graph algorithm. The result is a better Systems Biology analysis of your data that provides improved biomarker discovery, functional analysis and outcome prediction.

  • Data integration using multi-matrix factorization and multi-partite graph algorithms.
  • Multi-omics biomarker selection using PLS-DA, group LASSO, path scoring.
  • Integrative multi-omics clustering using SNF, parallel heatmaps, multi-layer t-SNE.
